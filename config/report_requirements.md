# AI Daily Report Requirements

Recipient name: yidan

Language: Simplified Chinese.

Tone:
- Professional and concise.
- Suitable for an executive or investor.
- Start with a 2-3 sentence internet-style comment before the formal report.
- The opening should be based on today's actual news, lively, concise, information-dense, with light wit.
- Avoid generic repeated lines, clickbait, memes, childish jokes, or exaggerated sarcasm.

Core sections:
1. 今日核心摘要
2. AI 技术角度
3. PE / 投资角度
4. 综合判断

今日核心摘要:
- This section should summarize today's actual information from the last 24 hours.
- Do not explain the report rules.
- Mention the main AI technology signal and the main PE/investment signal when available.
- Keep it concise, preferably one short paragraph or 2-3 dense sentences.
- The summary must be specific to the selected daily news and should not reuse yesterday's sentence structure.

Each news item should include:
- 主题
- 要点
- 影响
- 来源时间
- 来源

Freshness requirement:
- Only use news published or updated within the last 24 hours.
- Every selected item must include source_published_at.
- If there is not enough reliable information in a section within the last 24 hours, say so instead of filling with older news.

Topics to prioritize:
- Agentic AI
- AGI debate
- AI safety and regulation
- Silicon Valley AI products, platforms, model labs, chips, and developer ecosystem
- Wall Street AI stock outlook
- AI startup funding
- Private equity and enterprise AI deployment

Avoid:
- Long generic explanations
- Stale test data
- Hype without sources
- Markdown formatting in the email body
