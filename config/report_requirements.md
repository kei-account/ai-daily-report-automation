# AI Daily Report Requirements

Recipient name: yidan

Language: Simplified Chinese.

Tone:
- Professional and concise.
- Suitable for an executive or investor.
- Start with one short, lightly humorous summary before the formal report.
- The humor should be subtle, elegant, and brief.

Core sections:
1. 今日核心摘要
2. AI 技术角度
3. PE / 投资角度
4. 综合判断

Each news item should include:
- 主题
- 要点
- 影响
- 来源时间
- 来源

Freshness requirement:
- Only use news published or updated within the last 24 hours.
- Every selected item must include source_published_at.
- If there is not enough reliable information in a section within the last 24 hours, say so instead of filling with older news.

Topics to prioritize:
- Agentic AI
- AGI debate
- AI safety and regulation
- Silicon Valley AI products, platforms, model labs, chips, and developer ecosystem
- Wall Street AI stock outlook
- AI startup funding
- Private equity and enterprise AI deployment

Avoid:
- Long generic explanations
- Stale test data
- Hype without sources
- Markdown formatting in the email body
